Como Jogar?

O objetivo do jogo é tentar acertar o maior número de locais possíveis no tempo. Você pode olhar para todas as direções usando o mouse e, quando você estiver preparado para responder, bastar clicar no botão "Fazer chute!".

Você tem 4 chances e, caso você erre, a imagem ficará menos distorcida e você perderá uma chance. Se julgar necessário, clique na seta lateral para uma dica extra.

Quando terminar, você também poderá adicionar sua pontuação ao ranking. Boa sorte!
